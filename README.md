# Segundo Proyecto: AJAX / APIs
### Objetivo: 
Desarrollar una aplicación web dinámica e interactiva que consuma una API externa, presentando la información intuitiva y eficientemente.

### API elegida:
https://www.tvmaze.com/api

***

## Restricciones:
### Solo VanillaJs
Se evita el uso de jQuery para asentar las bases de JavaScript en sí.

### Realizar las solicitudes con Axios
Para mantener el código lo más limpio que se pueda, combinado con promise, y async/await.

### Subir el proyecto a algún servicio de alojamiento
Servicio elegido: Github Pages, por simplicidad.
